# chatbot_new
### Beschreibung
Ein Chatbot mit verschiedenen Oberflächen. Beispiel für objektorientierte Programmierung und die Verwendung von Frameworks.

Zusätzlich: Verwendung von Docker und CI/CD für Test und Verteilung.
